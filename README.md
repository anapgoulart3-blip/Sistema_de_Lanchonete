🥤 Cardápio de Bebidas (C#)
📌 Sobre

Este projeto é um programa simples em C# que simula um sistema de pedidos de bebidas em um terminal. O usuário pode escolher itens de um cardápio e o sistema calcula automaticamente o valor total do pedido.

⚙️ Funcionalidades
•Exibe um menu de bebidas com preços.
•Permite que o usuário escolha quantas bebidas quiser.
•Calcula o valor total do pedido automaticamente.
•Finaliza o pedido ao selecionar a opção “Sair/Finalizar”
Finaliza quando selecionar "5"

▶️ Como executar
Crie um projeto console:
dotnet new console
Cole o código no Program.cs
Execute:
dotnet run

📚 Conceitos Utilizados
Estrutura de repetição (do-while)
Estrutura condicional (switch-case)
Entrada de dados com Console.ReadLine
Conversão de dados (int.Parse)
Formatação de saída ({total:F2})

🎯 Objetivo

Praticar lógica de programação em C#.
